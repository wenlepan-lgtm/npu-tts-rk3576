# RKNN API Header

Place `rknn_api.h` here to enable NPU support.

Download from: https://github.com/rockchip-linux/rknpu2/blob/master/runtime/Android/librknn_api/include/rknn_api.h

```
cpp/rknn/rknn_api.h
```

If this header is missing, the build will use stub functions (NPU won't work, but the app compiles).
