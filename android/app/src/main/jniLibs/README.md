# jniLibs — Native Libraries

Place the following `.so` files here before building:

```
jniLibs/
└── arm64-v8a/
    ├── libonnxruntime.so    # ONNX Runtime for Android (from https://github.com/microsoft/onnxruntime/releases)
    ├── librknnrt.so         # RKNN Runtime for RK3576 (from https://github.com/rockchip-linux/rknpu2)
    └── libc++_shared.so     # NDK C++ shared library (auto-generated by Gradle)
```

## Where to get

### libonnxruntime.so
Download from [ONNX Runtime releases](https://github.com/microsoft/onnxruntime/releases):
```
onnxruntime-android-1.17.0.zip → jni/arm64-v8a/libonnxruntime.so
```

### librknnrt.so
Download from [RKNPU2](https://github.com/rockchip-linux/rknpu2):
```
rknpu2/runtime/Android/librknn_api/arm64-v8a/librknnrt.so
```

## Alternative: Use pre-built AAR

If you have `sherpa-onnx-1.12.24-rknn.aar`, it bundles both libraries. Extract with:
```bash
unzip sherpa-onnx-1.12.24-rknn.aar -d aar_extracted
cp aar_extracted/jni/arm64-v8a/*.so app/src/main/jniLibs/arm64-v8a/
```
